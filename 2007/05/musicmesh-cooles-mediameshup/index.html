musicmesh – Cooles Mediameshup

May 09, 2007 in COMPUTER SCIENCE • WEB2.0
dominican republic final cut express florida meshup musik national weather service snow songs anhören states video web web2.0 youtube
1 min read

Für jeden der schon immer mal das Video zu einem Audiotrack sehen wollte sollte sich einmal www.musicmesh.net ansehen. Dort kann man nach beliebigen Musikern suchen und sich danach ihre Songs anhören. Das coole dabei, es wird automatisch ein Musikvideo auf youtube gesucht und abgespielt! Sehr coole Sache!

Metadaten über Musiker, Tracks, Alben, etc… werden per Audioscrobbler Web Services API von www.last.fm bezogen, passende Videos werden wie gesagt über die API von YouTube gesucht und eingebunden! Auf jeden Fall einen Klick wert!
